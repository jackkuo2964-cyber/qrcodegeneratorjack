常見問題

QR Code 產生器是免費的嗎？

是的，這個 QR Code 產生器完全免費使用，不需要註冊帳號，所有運算都在瀏覽器本機完成，不會上傳或儲存任何輸入資料。

可以產生哪些類型的 QR Code？

支援三種內容類型：網址（可加上 UTM 行銷追蹤參數）、電話號碼（掃描後直接撥號）、以及任意文字或數字字串。

什麼是 UTM 參數，為什麼 QR Code 需要設定？

UTM 參數是附加在網址後面的追蹤代碼，包含來源（utm_source）、媒介（utm_medium）、活動名稱（utm_campaign）與內容（utm_content），可在 Google Analytics 等分析工具中追蹤這個 QR Code 是從哪個通路、哪個素材被掃描，常用於評估行銷活動成效。

下載的 QR Code 圖檔可以印刷使用嗎？

可以，提供 200px 到 1000px 共四種尺寸，其中 1000×1000px 為印刷用解析度，適合用於海報、DM、商品包裝等實體印刷物。
